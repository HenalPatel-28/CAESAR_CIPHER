# Caesar Cipher Tool

A Python-based GUI application that demonstrates the implementation of the Caesar Cipher encryption technique using the Tkinter library. This project allows users to encrypt and decrypt messages using a customizable shift value and also provides a brute-force attack feature to analyze encrypted text.

## Features

* Encrypt plaintext using the Caesar Cipher algorithm.
* Decrypt ciphertext using a specified shift value.
* Brute-force decryption by trying all possible shift values (1–25).
* User-friendly graphical interface built with Tkinter.
* Supports uppercase letters, lowercase letters, and special characters.
* Input validation and error handling for a smooth user experience.

## Technologies Used

* Python 3
* Tkinter (GUI Framework)

## Learning Objectives

* Understand classical cryptography concepts.
* Learn Caesar Cipher encryption and decryption techniques.
* Explore brute-force attack methods on simple substitution ciphers.
* Develop desktop GUI applications using Python Tkinter.

## How It Works

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted by a fixed number of positions in the alphabet. The application allows users to choose the shift value, encrypt messages, decrypt encrypted text, and test all possible decryptions using the brute-force feature.
